# Data-Science-Projects-Codes

This repository is a comprehensive collection of codes, notebooks, and projects exploring fundamental to advanced concepts in Data Science, Machine Learning prerequisites, and essential tools like Python, Pandas, NumPy, SQL, and advanced visualization techniques.

Whether you are a beginner looking to build a strong foundation or an experienced practitioner seeking quick references, this repository offers well-organized and practical examples.

## Key Learning Areas

The projects and codes in this repository are divided into the following core categories:

### I. Python & Programming Fundamentals

This section covers the building blocks of coding necessary for data manipulation and analysis.

| Folder | Description |
| :--- | :--- |
| **`1 - Unlocking the Power of Variable and Logic`** | Introduction to fundamental programming concepts: variables, data types, and logical operations. |
| **`2 -Loops Strings and Smart Operations`** | Mastering loops (`for`, `while`), string manipulation, and efficient coding techniques. |
| **`3 -Understanding Data Structures and Functions`** | Deep dive into Python's built-in data structures (lists, dicts, sets, tuples) and creating reusable functions. |
| **`4 - Advance Data Structures and Comparison`** | Exploring more complex data structures and methods for efficient data comparison and manipulation. |
| **`5 - Introduction to OOP Concepts`** | Introduction to Object-Oriented Programming (OOP) principles like classes and objects. |
| **`6 - Inheritance and Advanced OOP Features`** | Advanced OOP topics, including inheritance, polymorphism, and encapsulation. |

### II. Data Manipulation & Analysis (Pandas & NumPy)

Focus on the industry-standard libraries for handling and processing data.

| Folder | Description |
| :--- | :--- |

| **`10 - Introduction to Pandas`** | Getting started with Pandas: creating DataFrames and Series, basic data loading, and inspection. |
| **`11 - Pandas Data Manipulation Basics`** | Core data manipulation tasks: indexing, selecting, filtering, and cleaning data using Pandas. |
| **`12 - Numpy Attributes and Advanced Pandas ...`** | Combining NumPy's array attributes with advanced features in Pandas for high-performance data operations. |

### III. Statistics & Modeling Prerequisites

The mathematical and statistical foundation required for data analysis and machine learning.
